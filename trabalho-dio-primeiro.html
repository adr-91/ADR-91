A Historia do HTML

HTML (Markup Language) é a linguagem padrão para a criação de páginas na web

O Início do HTML

No final dos anos 80, o físico britânico Berners-Lee trabalhava no CERN e estava buscando uma maneira de compartilhar informações científicas entre diferentes instituições no ano de 1989, ele propôs um sistema de gerenciamento de informações que utilizava o conceito de "hipertexto". Este sistema viria a se tornar a World Wide Web

.

A primeira versão

Em 1991, Berners-Lee desenvolveu o HTML 1.0.Essa primeira versão do HTML era bastante limitada, mas serviu como a base para a criação das primeiras páginas web.

A Evolução do HTML: 1995 a 1999

À medida que a web crescia, também cresciam as demandas por uma linguagem mais rica e flexível. Em 1995, a HTML 2.0 foi lançada, incluindo melhorias significativas, como formulários para entrada de dados. Logo após, em 1997, a HTML 3.2

O Advento do HTML5: A Web Moderna

Durante a primeira década dos anos 2000, as limitações do HTML 4.01 começaram a se tornar evidentes, especialmente com o surgimento de novos tipos de conteúdo multimídia e interatividade. O consórcio WHATWG. (Web Hypertext Application Technology Working Group). começou a trabalhar em uma nova especificação, que culminaria no lançamento do HTML5 em 2014

O HTML5 foi um marco na evolução da web, trazendo inúmeras melhorias:

"O HTML5 é mais que uma evolução, é uma revolução na forma como interagimos com a web."
HTML Hoje: A Base da Web Contemporânea

Hoje, o HTML continua a ser a espinha dorsal da web, suportado por CSS e JavaScript, que juntos permitem a criação de sites e aplicações web altamente interativos e esteticamente agradáveis
